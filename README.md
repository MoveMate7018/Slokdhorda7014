# MoveMate - Fast & Reliable Goods Transportation

MoveMate is a modern web-based platform for goods transportation services, providing users with an easy way to book and manage transportation of goods.

## Features

- **Homepage**: Introduction to MoveMate services with modern UI
- **About Us**: Information about the company and its mission
- **Services**: Detailed view of transportation services offered
- **Booking**: Easy-to-use booking form for scheduling transportation
- **Contact**: Contact page for customer inquiries
- **Customer Care**: Dedicated customer support page
- **Login/Registration**: User authentication system

## Getting Started

1. Clone the repository
2. Open `Movemate/homepage.html` in your web browser
3. Navigate through the application using the sidebar menu

## Pages

- **Home**: Main landing page with service overview
- **About**: Company information and mission
- **Services**: Available transportation services
- **Booking**: Book a transportation service
- **Contact**: Get in touch with us
- **Customer Care**: Support and help desk
- **Login/Register**: User account management
